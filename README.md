# EduSense: AI-Powered Learning Assistant

Welcome to my repository for the Seasons of Code (SOC) EduSense project. This repository contains my weekly assignments, code implementations, and technical reports tracking my progress in building an automated AI learning assistant.

The ultimate goal of EduSense is to process raw educational audio recordings or long technical documents, transcribe them accurately, and generate concise, high-density study summaries using deep learning and natural language processing.

## Repository Contents

* **Week 1:** Data Cleaning and Preprocessing (Pandas, NumPy, Matplotlib, Outlier Handling, PCA)

* **Week 2:** Deep Learning Fundamentals (PyTorch, Tensors, Training Loops, Linear Regression)

* **Week 3:** Audio Processing Frontend (Bandpass Filtering, Spectral Denoising, Silero VAD, OpenAI Whisper Transcription)

* **Week 4:** Text Summarization Backend (Google Pegasus Abstractive Summarization, Hyperparameter Tuning, Extractive LSA, ROUGE Evaluation)

* **Midterm_Report.md:** A summary of my foundational learnings and progress from the first half of the project.

* **Progress_Report.md:** A detailed technical breakdown of the summarization models, comparative analysis, and evaluation metrics implemented in Week 4.

## Tools and Technologies Used

* **Core Data Science:** Python, NumPy, Pandas, Matplotlib, SciPy

* **Deep Learning Frameworks:** PyTorch, Hugging Face Transformers

* **Audio Engineering:** Librosa, Silero-VAD, OpenAI Whisper

* **NLP & Summarization:** Google Pegasus (pegasus-xsum, pegasus-arxiv), Sumy, NLTK, ROUGE-score
